# SmartGate AI 🛡️
## The Future of Residential Intelligence & Security

**SmartGate AI** is a cutting-edge, comprehensive management platform designed for modern residential communities. By leveraging the power of **Google Gemini AI** and **Maps Grounding**, it transforms traditional property management into a proactive, intelligent, and secure experience.

---

### 🚀 Key Features

*   **🛡️ AI Surveillance Dashboard**: Real-time object detection (People/Vehicles) with live camera feeds and interactive OSD overlays.
*   **🎫 Visitor Management 2.0**: Secure pre-registration for guests and service providers with dynamic QR code generation for seamless entry.
*   **📍 Neighborhood Intelligence**: Deep, real-time insights into surrounding amenities and services using AI-driven location grounding.
*   **🐾 Pet Tracking AI**: Intelligent monitoring of neighborhood pets with virtual markers, last-seen tracking, and geo-fenced safety zones.
*   **⚡ Utility Analytics**: Advanced data visualization for community energy usage trends and billing management.
*   **🏠 Smart Home Hub**: Integrated control interface for residential IoT devices like lighting, locks, and climate control.

---

### 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) with [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **AI Core**: [Google Gemini 3.1 Flash](https://ai.google.dev/)
- **Intelligence**: Google Maps Grounding via `@google/genai`
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Motion for React](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

### 📦 Setup & Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd smartgate-ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root directory or configure your platform's environment variables:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Launch the development server**
   ```bash
   npm run dev
   ```
